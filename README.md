# ShopEase - Shopping Landing Page

A responsive and user-friendly shopping landing page built with **Next.js**, **TypeScript**, and **Tailwind CSS**.  
This project showcases essential e-commerce features such as product listing, search, filtering, and a shopping cart system.

-

## Features

- Product listing with image, name, and price.
- Search products by name.
- Filter products by price range.
- Filter products by category.
- Add to cart with quantity support.
- Remove items or clear the cart.
- View total price and simulate checkout.
- Cart data persisted in `localStorage`.
- Smooth transitions using `framer-motion`.
- Toast notifications with `react-hot-toast`.
- Fully responsive and mobile-first design.

-

## Tech Stack

| Technology           | Purpose                        |
| -------------------- | ------------------------------ |
| Next.js (App Router) | Framework for React app        |
| TypeScript           | Type-safe development          |
| Tailwind CSS         | Utility-first styling          |
| React Context API    | Cart state management          |
| Framer Motion        | Animations & transitions       |
| React Hot Toast      | Notifications (Toast messages) |
